# Embedded Systems & IoT Portfolio

**Description**
This repository contains my development work with microcoltrolers (Arduino, ESP32) and embedded systems.
With a focus on AgriTech solutions such as automated monitoring and academic research.

**Folder Structure**

* `/Backup e projetos do RESCAAT` - **Research Projects.**
   Contains source codes developed for **Project RESCAAT**. Includes logic for IoT sensors used to monitor environmental conditions on a small scale in the Caatinga (semi-arid biome).

* `/Projetos em desenvolvimento` - **Academic & Ongoing Projects.**
   Contains personal prototypes (including SmartCrop) and practical assignments developed for the Science & Technology course at UFERSA.

**Tech Stack**
* **Hardware:** Arduino (Uno/Mega/Nano), ESP32, Sensors (DS18B20, Soil Moisture).
* **Languages:** C/C++ (Firmware), Python.
* **Tools:** VS Code, Git.
